# Process Docs

These docs define the generic Flywheel workflow contract.

## Core Docs
- `CYCLE_NAVIGATION.md`
- `STAGE_CONTRACTS.md`
- `STAGE_EXIT_GATES.md`
- `HANDOFF_EXPECTATIONS.md`
- `VERIFICATION_PATTERNS.md`

All workflow locations are resolved through `flywheel.yaml`.

In embedded installs, interpret those locations relative to the directory containing `flywheel.yaml`, not the repo root.
